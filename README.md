# Guess Golleague

## Запуск проекта:

1. Вставить в main.py токен (см. пример)

```ini
bot = TeleBot('0000000000:AAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAAA')
```

### Необходимо установить библиотеки из файла requirements.txt (установить их можно введя команду):

```comandline
pip install -r requirements.txt
```

### Необходимо инициализировать базу данных:

```comandline
python utils.py
```

### Запуск 

```comandline
python main.py
```

### Примечание
Каждое фото должно храниться в папке photos и назваться так, как вы хотите, чтобы выглядели кнопки выбора

Пример: "Иван Иванов.jpg"
